# 🚧 Smart Helmet Detection System

## 🔍 Overview

This project detects whether a rider is wearing a helmet using YOLOv8.
It helps in identifying safety violations in real-time and can be extended to smart traffic monitoring systems.

---

## 🛠️ Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* Matplotlib

---

## ⚙️ Features

* Detect rider
* Detect helmet
* Identify helmet / no helmet
* Safety violation detection

---

## 📂 Dataset

Dataset used from Kaggle containing:

* Rider
* With Helmet
* Without Helmet
* Number Plate

---

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the notebook:
   Open `smart-helmet-detection.ipynb`
