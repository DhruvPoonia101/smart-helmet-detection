# 🚧 Smart Helmet Detection System

## 🔍 Overview
This project detects whether a rider is wearing a helmet using YOLOv8.

## 🛠️ Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Matplotlib

## ⚙️ Features
- Detect rider
- Detect helmet / no helmet
- Safety violation detection

## 📂 Dataset
Dataset used from Kaggle

## 🚀 How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run notebook:
   Open smart-helmet-detection.ipynb
